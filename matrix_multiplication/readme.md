# Matrix Multiplication

**Program Name:** Matrix Multiplication

**Problem Statement:** Multiply two matrices using dynamic memory allocation.

**Input Example:**  
Matrix A: 2x3 → {{1,2,3},{4,5,6}}  
Matrix B: 3x2 → {{7,8},{9,10},{11,12}}

**Output Example:**  
Resulting Matrix:  
58 64  
139 154

**Complexity:**  
Time: O(n^3)  
Space: O(n^2)
